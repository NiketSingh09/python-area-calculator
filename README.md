📐 Area Calculator (Python)

A simple command-line based Python program that calculates the area of different geometric shapes based on user input.

This project demonstrates structured programming using functions, conditional statements, loops, and user interaction.

🚀 Features

Interactive CLI-based interface

Calculates area of:

Circle

Rectangle

Triangle

Uses functions for modular code structure

Handles user input dynamically

Beginner-friendly and expandable design

🧠 Concepts Used

Functions

if / elif / else statements

Loops

User input handling

Basic mathematical formulas

🛠️ How It Works

The program asks the user to enter a shape.

Based on the shape selected, it requests the required dimensions.

It calculates the area using the appropriate mathematical formula.

The result is displayed instantly.
